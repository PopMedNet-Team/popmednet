This Distributed Query Tool and Portal is for use by authorized users. All use must be related to MDPHnet activities, and users must clearly describe the purpose of their query and the intended use of the query results. Data from the Distributed Query Tool and Portal are intended for public health surveillance activities and cannot be published without prior approval of the data partners. External recipients of data from this tool must agree to these Terms and Conditions, and all information shared must be marked CONFIDENTIAL and sent securely. Use of the tool is monitored and audited. User privileges may be revoked at any time.
